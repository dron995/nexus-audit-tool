# Nexus Security Audit Tool

Инструмент для аудита безопасности репозиториев Nexus OSS.

## Возможности

- Проверка анонимного доступа ко всем типам репозиториев (hosted, proxy, group)
- Поддержка различных форматов артефактов (Docker, Maven, NPM и др.)
- Обработка самоподписанных SSL сертификатов
- Группировка отчетов по типам артефактов и репозиториев
- Генерация отчетов в форматах: HTML, JSON, текстовый
- Отправка отчетов по email
- Генерация Prometheus метрик
- Конфигурирование через YAML файл
